# YouTube Music Setup

This section covers the Google OAuth setup required for private YouTube Music access.

---

## Overview

To use the YouTube Music integration, you need:

1. Google OAuth credentials for a web application
2. Exact redirect URI matching
3. Connected users for private playlists, liked songs, and library albums
